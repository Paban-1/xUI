How to Use xUI Components

  1. Browse: Scroll through the component gallery on the home page.
  2. Preview: Each component displays a live preview of what it looks like.
  3. Copy Code: Click “Copy HTML” or “Copy JS” to copy the respective code.
  4. Paste and Use: Paste the code into your project and see it in action.

Tip:

You can modify styles using Tailwind utility classes and customize JS behavior to fit your app.
